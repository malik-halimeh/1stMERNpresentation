Single-narrative demo ordering

  4. Staff confirm → ship → deliver in the admin queue; customer's bell + email fire with each step. Haya drives, Malik narrates the state machine.

  7. Analytics + audit trail: revenue vs purchase-spend charts (margin story); try to edit an audit log in the shell — watch it throw. Malik + Mahmod.
  8. Security arc: "our audit found these — here's the fix, and here's test_fixes.ts proving it" (20/20 run live if time allows). Malik.
